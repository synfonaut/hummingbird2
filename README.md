# Hummingbird2
> A Bitcoin Application Framework

![logo](./logo.png)

Hummingbird2 is a light wrapper around the best tools in Bitcoin:

* bitbus
* bitsocket
* bitquery
* bitfs
* shapeshifter.js
* bmap
* 21e8
* boostpow
* twetch
* bitsv

Release is alpha, better documentation coming. Read the source for now.
